Dynamic Code Injection

Dynamic Code Injection is a tool that allows you to increase your productivity by decreasing time that you generally wasting when waiting for project recompilation and restart on device. Main idea behind this tool is to allow you to change his code and see new results immediately. It is not matter of recompilation it is matter of decreasing time interval from code or resources changes to actual result. If you still don't understand what this tool for, just look to it's features.
