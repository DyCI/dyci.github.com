Dynamic Code Injection

Change your code and see the results right away. It's that simple.

Dynamic Code Injection will boost your productivity by saving the time you waste while waiting for project to recompile and restart on simulator. In other words, no more standby between altering your code or resources and seeing the actual result.

And we promise, you'll never miss that progress bar and "Compiling 32 of 32489 resources" again:)
